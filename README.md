# Portale Open Data - Comune di Salle

## Descrizione

In questa pagina sono pubblicati i dati aperti relativi al Comune di Salle, come previsto dalle normative in materia di trasparenza e riutilizzo dei dati pubblici.

## Contenuto

- [Struttura Repository](#struttura-repository)
- [Formato Dati](#formato-dati)
- [Aggiornamento Dati](#aggiornamento-dati)
- [Licenza](#licenza)

## Struttura Repository

I dataset sono contenuti all’interno della cartella `dataset`.

Ogni dataset è organizzato in una specifica sottocartella nominata come il dataset stesso.

All’interno di ogni cartella sono presenti:

- file dei dati
- documentazione descrittiva
- metadati associati al dataset
- informazioni su aggiornamento e frequenza di pubblicazione

## Formato Dati

I dati sono pubblicati in formati aperti e standard, tra cui:

- CSV per dati tabellari
- GeoJSON per dati geografici
- GeoPackage per dati territoriali

Ulteriori formati possono essere utilizzati laddove necessario.

## Aggiornamento Dati

I dati sono aggiornati secondo le tempistiche definite per ciascun dataset.

La frequenza può essere:

- periodica
- annuale
- su aggiornamento dell’ente

## Licenza

Salvo diversa indicazione, i dati sono rilasciati con licenza aperta che ne consente il riuso.
